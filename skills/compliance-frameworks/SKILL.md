# Compliance Frameworks

## When to Apply
- GDPR: Any application processing EU personal data
- WCAG: All web applications (especially government, enterprise, public-facing)
- SOX: Financial reporting systems in publicly traded companies

## General Approach
1. Identify which regulations apply
2. Map requirements to current implementation
3. Identify gaps
4. Prioritize remediation by risk
5. Document compliance status

For GDPR specifics, see [gdpr.md](gdpr.md).
