# Research Methodology

Conventions for conducting rigorous, actionable research.

## Source Classification

- **Primary sources**: Original data, experiments, firsthand accounts, official documentation
- **Secondary sources**: Analysis, reviews, summaries of primary sources
- Always prefer primary sources; use secondary sources for context and synthesis

## Source Credibility Evaluation

Evaluate every source against these criteria:

- **Recency**: Is the information current? Technology sources older than 2 years need verification
- **Authority**: Is the author/organization credible in this domain?
- **Methodology**: How was the information gathered? Is the methodology sound?
- **Bias**: Does the source have commercial, ideological, or institutional bias?
- **Corroboration**: Do other independent sources support the claims?

See [Source Evaluation](source-evaluation.md) for the full framework.

## Literature Review Organization

Organize findings **thematically, not source-by-source**. Group by:

1. Topic or question addressed
2. Consensus areas (where sources agree)
3. Contested areas (where sources disagree)
4. Gaps (questions no source addresses adequately)

## Synthesis Patterns

Research value comes from synthesis — connecting findings across sources to produce insights none provide individually.

- **Cross-source triangulation**: Validate claims by finding independent confirmation
- **Gap identification**: Map what is known vs unknown; identify where evidence is thin
- **Contradiction mapping**: When sources disagree, analyze why and assess which is more credible
- **Confidence levels**: Assign confidence (high/medium/low) based on evidence strength

See [Synthesis Patterns](synthesis-patterns.md) for detailed patterns.

## Report Structure

1. **Executive summary** — Key findings and recommendations in 3-5 bullets
2. **Methodology** — What was researched, how, and what sources were used
3. **Findings** — Organized thematically with inline citations
4. **Analysis** — Synthesis, patterns, trade-offs, and implications
5. **Recommendations** — Actionable next steps tied to findings
6. **Limitations** — What the research does not cover, confidence caveats
7. **References** — All sources cited, with access dates for web sources

## Citation Standards

- Cite inline using `[Author/Org, Year]` or `[Source Name]` format
- Collect all references in a dedicated section at the end
- Include URLs and access dates for web sources
- Distinguish between facts (cited) and analysis (your synthesis)
