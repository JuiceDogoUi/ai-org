# Social Media Guide

## Platform Conventions

### Twitter / X
- **Length:** 280 characters max; aim for punchy and complete
- **Format:** Hook → Insight → CTA or question
- **Threads:** First tweet must stand alone; each tweet should add value independently
- **Visuals:** Code screenshots, diagrams, and short videos perform well
- **Hashtags:** 1-2 max, only if genuinely relevant

### LinkedIn
- **Length:** First 3 lines are visible before "see more" — make them count
- **Format:** Personal insight → Supporting evidence → Takeaway
- **Tone:** Professional but human — avoid corporate-speak
- **Visuals:** Carousels and infographics drive high engagement
- **Hashtags:** 3-5 relevant to your industry

### Instagram
- **Length:** Caption supports the visual — keep it scannable
- **Format:** Visual-first; caption adds context and CTA
- **Stories:** Behind-the-scenes, polls, Q&A for engagement
- **Reels:** Short, valuable, and hook within 1 second
- **Hashtags:** 5-10, mix of broad and niche

## Post Templates

### Share an Article
```
[Hook — why this matters to the reader]

[1-2 sentence summary of the key insight]

[Link]

[Question to drive engagement]
```

### Share a Tip
```
[Specific tip in one sentence]

[Why it works / when to use it]

[Optional: screenshot or example]
```

### Announce a Feature
```
[What's new — in plain language]

[What it means for the user]

[Link to learn more or try it]
```

## Post Checklist

- [ ] First line hooks attention
- [ ] One clear idea per post
- [ ] Written for the specific platform (not copy-pasted across all)
- [ ] Visual included when possible
- [ ] Links are working and trackable
- [ ] Engagement prompt (question, opinion request, or CTA)
- [ ] Proofread — no typos
- [ ] Scheduled at an appropriate time for the audience
