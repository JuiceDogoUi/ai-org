# UX Copy Patterns

## Confirmation Dialogs
- Title: State the action ("Delete this project?")
- Body: Explain the consequence ("This will permanently remove all files and data.")
- Primary action: Matches the title verb ("Delete")
- Secondary action: "Cancel" (not "No" or "Go back")

## Success Messages
- Confirm what happened: "Project created successfully"
- Suggest next step if relevant: "Project created. Add your first task to get started."
- Auto-dismiss after 3-5 seconds for non-critical messages

## Form Labels & Hints
- Label: What the field is ("Email address")
- Placeholder: Example format ("name@company.com")
- Hint: Constraints or context ("We'll send a verification link")
- Never use placeholder as the only label

## Tooltips
- Explain what something does, not what it is
- Keep under 150 characters
- Show on hover/focus, not on click
