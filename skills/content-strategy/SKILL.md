# Content Strategy

## Article Structure
1. **Hook**: Open with a problem, question, or surprising fact
2. **Context**: Why this matters to the reader
3. **Body**: Deliver value with clear sections and examples
4. **Conclusion**: Summarize key takeaways, include CTA

## Writing Style
- Write at an 8th-grade reading level for broad accessibility
- Use short paragraphs (2-4 sentences)
- Use subheadings every 200-300 words for scannability
- Include one concrete example per major point
- Avoid jargon without explanation

## SEO Basics
- Include primary keyword in title, first paragraph, and 1-2 subheadings
- Write meta description under 160 characters with keyword
- Use descriptive, keyword-rich alt text on images
- Internal link to 2-3 related articles
- External link to 1-2 authoritative sources
